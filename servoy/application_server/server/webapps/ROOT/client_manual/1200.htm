Reducing the found set

Reducing the found set - also called find in find - is a very useful way to search within your current found set.

To search within your current found set:

  • Choose Select > Find. OR

Press the keyboard shortcut CTRL-F / CMD-F.

  • Type in your find request.
  • Choose Select > Reduce Search.
