Servoy modes

Servoy Client has two modes of operation:

  • Find mode.
  • Data (ready) mode.

Find mode

Find mode can be used to:

  • Search for data in zero or more records in a single Find request.
  • Search for data in zero or more records in multiple Find requests.
  • Search for data in zero or more records based on an
    extended search.
  • Search for data in zero or more records based on a
    reduced search.

For more detail on finding data, refer to the section on
Finding data in the chapter Getting Started.

Data (ready) mode

Data (ready) mode enables you to:

  • View all data in a form for a found set of records.
  • Omit (from view only) certain records in a found set.
  • Show all omitted records in a form.
  • View all the records that are not in a found set.

  • Add, modify and delete data and/or records.

For more detail, see Editing data.

  • Sort data in a form for a found set of records.

See the section on Sorting data in the
Servoy Client User's Guide.

  • Preview the data from one or more forms.
  • Print the data in one or more forms as a report.

See the section on Printing for more information.

  • Import and export data.

See the chapter on Importing and Exporting Data.
