Relooking up data

The solution that you are using may enable you to relookup data -
update data based on related information - in one or more editable
fields on a form for a found set of records.

To relookup data in an editable field for one or more records:

  • Search for the found set of records you want to relookup data for.

For more information, see the section called Finding data.

  • Make sure you are viewing the found set of records in Data (ready) mode - not Find mode.
  • Click or TAB into the specific field you want to relookup data for.

(You can select the desired field in any record of the found set.)

  • Choose Select > Relookup.


  • The data in the editable field you selected will be updated for
    all records in the found set.

NOTE: For information on Relookup and relationships, refer to the sections on Methods, Form functions, Methods, Form relations, and Defining relations in the Servoy Developer User's Guide.
