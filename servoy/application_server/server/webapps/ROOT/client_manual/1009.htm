Introduction to importing and exporting

Servoy supports the importing and exporting of:

  • Text files.
  • Excel files.

Supported text formats

To import to a text file or export data as a text file, you can use the:

  • Tab separator.
  • Semi-colon separator.
  • Comma separator.
  • Space separator.
  • Custom separator.

NOTE: The Importing and exporting feature(s) are only available if your administrator has included the import/export plugin(s) on the Servoy Server
