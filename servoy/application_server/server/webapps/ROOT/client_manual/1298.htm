Opening a solution

To open a solution:

  • Choose File > Open. OR

Press CTRL-O / CMD-O. OR

Click on the Open Solution tool in the Edit toolbar.

  • Choose the solution you want to open from the Select Solution dialog.
  • Press the OK button to open the solution.
  • You should see the spinning Servoy progress screen as the solution you selected opens:

NOTE: You may be presented with the Select solution dialog when Servoy first opens. Follow the two steps listed just above the note.
