Deleting records

You can delete:

  • The current record.
  • All records in the current found set.

To delete the currently active record:

  • Make sure that you are not in Find mode.
  • Choose Select > Delete record OR
  • Use the keyboard shortcut CTRL-E / CMD-E.

To delete all records in the current found set:

  • Make sure that you are not in Find mode.
  • Choose Select > Delete All.

NOTE: One or more records may be set to read-only or your permissions might not allow you to delete the record(s).

IMPORTANT: There is no Undo command available when you delete one or more records.
