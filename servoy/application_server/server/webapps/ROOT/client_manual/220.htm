Editing Data

Data can be edited in Servoy in Data (ready) mode by:

  • Adding records.
  • Modifying records.
  • Duplicating records.
  • Deleting records.

NOTE: Drag & Drop is supported in Data (ready) mode (not in Find mode) within the same record for text fields when Java JRE 1.6.x (or higher) is installed.
