Starting Servoy Smart Client

Servoy Smart Client is installed automatically so it does not have to be installed.

You can start Servoy Smart Client:

  • From the Servoy Server web page that shows the link
    launch servoy client.

OR

  • By typing in the URL address for Servoy Smart Client.

To launch Servoy Smart Client from the Servoy Server web page:

  • Open up your web browser.
  • Enter the URL or IP address of the server.

After a default installation, the server runs on the port 8080.

For example:

If Servoy Server is running on a server computer called "CentralServer", then type the following text in the Address bar of your web browser:

http://centralserver:8080/

You will see the following screen:

  • Click the link launch servoy client in the center of the page.
  • You will see this page displayed:

  • Then you should see this screen:

Once the necessary files have been downoaded, you should see this screen:

  • Servoy Smart Client will start and enable you to open your solution.

For more detail, see Opening a solution.

NOTE: The screens you see when launching Servoy may vary slightly - especially after the first time you launch Servoy; and depending on the speed of the computer you are using.

To launch Servoy Smart Client by typing in a URL address:

  • Open up your web browser.
  • Enter the URL address for Servoy Smart Client.

(You may need to check with your database Administrator if you don't know the URL address for Servoy Smart Client.)

For example, to start the Servoy CRM Demo, enter:

http://localhost:8080/servoy-client/servoy_client_crm.jnlp

  • Press the ENTER key or RETURN key on your keyboard to
    launch Servoy Smart Client.
