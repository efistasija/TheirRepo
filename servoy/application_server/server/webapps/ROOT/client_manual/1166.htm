Look And Feel

The Look And Feel preferences can be set here.

Look and feel:
Servoy supports multiple Look And Feels (LAF). By default, the following Look And Feels are supported:

  • Metal (LAF)
  • Kunststoff (LAF)
  • Your operating system Look And Feel (if available)

Theme:
Variations on a Look And Feel (if available). The themes available for the Look And Feel (LAF) that is selected are displayed in the Theme pull-down menu.

By default, the following themes are included with the Kunststoff Look And Feel:

  • Desktop Theme
  • Notebook Theme
  • Presentation Theme
  • Kunststoff Theme

Default font:
In the Look And Feel tab, you can also set the default font that is used by Servoy Developer. The fonts available on your system are displayed in the Default font pull-down menu.
