Services

The Services tab is used to configure the Servoy application service.

Start with two way socket for using clients behind firewall
Check this box if Servoy Server is running behind a firewall or clients on a different network need to connect to the server. This box is checked by default when Servoy is installed.

NOTE: If you make any changes to this screen, you need to restart the Servoy Client application in order to activate the changes.
