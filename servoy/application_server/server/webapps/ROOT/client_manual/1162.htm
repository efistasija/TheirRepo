Application preferences

All Servoy Developer preferences are set in the Application Preferences tabs. If plugins have preferences (like the Agent plugin), they can also have a tab here.

To set Application preferences:

  • Choose Edit > Preferences.

  • Select an Application Preferences tab.

The following preferences can be set:

  • General
  • Look And Feel
  • Locale
  • Services
  • Agent

NOTE: All application preferences require Servoy to be restarted in order for any changes to take effect.
