About this book

The following book is the Servoy Client User's Guide.

This book can be purchased from the Servoy Store and/or can be downloaded online from the Developer/Documentation page of Servoy.

Servoy Client User's Guide

The Servoy Client User's Guide contains all the information you need to use your Servoy solution over a local area network; a wide area network; or even over the internet. All you need is a web browser and the correct version of Java on your machine. Servoy can be started from a browser and will automatically download any software that is needed directly to your machine. The Servoy Client User's guide can help you to view, edit, find, print, and import data - depending on your solution.

Also available for purchase from the Servoy Store and downloadable as two pdf files is The Servoy Developer book.

The Servoy Developer book is made up of two (2) volumes:

  • Volume 1: User's Guide
  • Volume 2: Reference Guide

Servoy Developer User's Guide

The Servoy Developer User's Guide contains the information needed to help you get started and become experienced using Servoy: Servoy concepts; finding, viewing and editing data; step-by-step how to create a Servoy solution; Servoy forms and elements; Servoy styles; printing; Servoy security; importing and exporting data; creating and connecting to a database; Servoy modules; i18n; deployment using Servoy Server and Servoy Client; and much more.

Servoy Developer Reference Guide

The Servoy Developer Reference Guide contains a complete list of functions and runtime properties that can be used in Servoy calculations and/or methods, including a description and example(s) for each property and function; plugins; Servoy Runtime; a list of JavaScript escape codes; a table of Servoy reserved words; the JavaScript reserved words list; as well as other reference information.
