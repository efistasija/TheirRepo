Duplicating records

To create a record with similar data to the currently active record, you can duplicate the current record.

To duplicate the currently active record:

  • Make sure that you are not in Find mode.
  • Choose Select > Duplicate record OR
  • Use the keyboard shortcut CTRL-D / CMD-D.

  • Click or TAB into any field(s) that you need to modify the data for
    in the duplicated record.
  • Make the appropriate changes to the data.
  • Choose Edit > Save data
    (to force the data to be saved before you exit the record).

When you exit the current row, the data will be saved to the backend database automatically.

When you duplicate a row (record), all data from the currently selected row will be duplicated except those fields that have an Auto enter sequence associated to them.
