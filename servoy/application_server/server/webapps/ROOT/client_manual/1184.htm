Viewing data

You can view data when you are in Data (ready) mode - not in Find mode.

In Servoy you can view data in a form in one of two ways:

  • Record view
  • List View

Viewing data in Record view

In Record view:

  • One record is displayed at a time.

Viewing data in List view

In List view:

  • More than one record is displayed at a time.

  • The current (active) record may show a vertical black line

(located to the left of the record).

  • The list of records may have a vertical scrollbar

(located on the right side of the active form when there are more records in
your current found set than will fit in the size of the currently active form).

  • The list of records may have a horizontal scrollbar

(located at the bottom of the active form when each record takes up more space
horizontally than will fit in the size of the currently active form).

Switching data views

Your solution may enable you to look at the data on a form as a single record or as a list of records.

To switch from Record view to List view:

  • Make sure you are in Data (ready) mode - not Find mode.
  • Choose View > View as List.

To switch from List view to Record view:

  • Make sure you are in Data (ready) mode - not Find mode.
  • Choose View > View as Record.

NOTE: Being able to view your data on a specific form in Record view or List view depends, in part, on the design of the solution you are using.

Servoy provides two ways for you to move easily between forms.

You can:

  • Use the built-in navigation arrows.
  • Use the Window menu.

Navigation arrows

To help you go to the next form or back to the previous form - similar to the buttons in your internet browser - Servoy has included built-in navigation arrows as part of the Edit toolbar.

Using the navigation arrows, you can go:

  • Back - to the previous form.
  • Forward - to the next form from the previous form.

To show the Edit toolbar:

  • Make sure you are not in Find mode.
  • Choose the menu View > Toolbars > Edit.

NOTE: For more information on toolbar options, see the section called Toolbars.

To go to the next form:

  • Make sure you are not in Find mode.
  • Click the Forward arrow on the Edit toolbar.

NOTE: The Forward arrow will be "grayed out" unless you have gone to more than two forms.

To go to the previous form:

  • Make sure you are not in Find mode.
  • Click the Back arrow on the Edit toolbar.

Servoy automatically keeps track of the last nine (9) forms you navigated to.

Navigating windows

You can also navigate between forms in Servoy by using the
Window menu.

To use the Window menu:

  • Make sure that you are Data (ready) mode - not in Find mode.
  • Click the Window menu in the Servoy Menu bar.
  • Move your mouse cursor to one of the forms in the list.
  • Click once to display the form you selected.

NOTE: The number and availability of forms included in the Window menu form list depends entirely on the Servoy solution you are using.

Servoy also provides a simple way to navigate to the next or previous record on the currently active form.

Navigating to the next or previous record

To view the data in the previous record:

  • Make sure you are in Data (ready) mode - not in Find mode.
  • Choose Select > Previous record. OR

Press the keyboard shortcut ALT-UP / OPTION-UP.

To view the data in the next record:

  • Make sure you are in Data (ready) mode - not in Find mode.
  • Choose Select > Next record. OR

Press the keyboard shortcut ALT-DOWN / OPTION-DOWN.

In Data (ready) mode you can:

  • Show all records in a found set on a form.
  • Omit (from view only) one or more records from the found set.
  • Show all of the omitted records.
  • Show all the records that are not in your found set.

NOTE: For more information on modes, see the section called Servoy modes earlier in this chapter.
