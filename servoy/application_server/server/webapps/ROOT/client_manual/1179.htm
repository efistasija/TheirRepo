Printing

With Servoy you can print:

  • All records.
  • A found set of records.
  • The current record.
  • A report with Subsummary parts.

Printing all records

To print all records:

  • Make sure you are in Data (ready) mode.
  • Choose Select > Show All. OR

Press the keyboard shortcut CTRL-J / CMD-J.

  • Choose File > Print Preview. OR

Press the keyboard shortcut CTRL-U / CMD-U.

  • Click once on the Print button.

Printing a found set of records

To print a found set of records:

  • Make sure you are in Data (ready) mode.
  • Perform a Find for the records you want to print.

(Your found set should contain the records you want to include.)

For more information on finding records, see the section on
Finding data in the Getting Started chapter.

  • Choose File > Print Preview. OR

Press the keyboard shortcut CTRL-U / CMD-U.

  • Click once on the Print button.

Printing the current record

To print the current record:

  • Make sure you are in Data (ready) mode.
  • Perform a find for the record you want to print.

For more information on finding records, see the section on Finding data in the Getting Started chapter.

  • Choose File > Print Preview. OR

Press the keyboard shortcut CTRL-U / CMD-U.

  • Click once on the Print button.

Printing a report with one or more Subsummary parts

To print a report with one or more subsummaries:

  • Sort the records by the break field (sort field) in
    each Subsummary part.

For detail on Sorting, see the section called Sorting data.

  • Choose Print Preview. OR

Press the keyboard shortcut CTRL-U / CMD-U.

  • Then select Print.

For more information on subsummary parts see the Forms, Parts section in the Servoy Developer User's Guide.
