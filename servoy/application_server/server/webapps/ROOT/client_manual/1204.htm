Toolbars

Servoy toolbars contain buttons to perform commonly-used functions. When you pass your mouse cursor over a toolbar button, you will briefly see a tooltip - a description of what the button is used for.

To show a toolbar:

  • Choose View > Toolbars.
  • Check the box next to the toolbar you want to show.

To hide a toolbar:

  • Choose View > Toolbars.
  • Uncheck the box next to the toolbar you want to hide.

You can reposition or change the order of the toolbars you are
viewing at the top of the screen.

To move a toolbar:

  • Click once on the vertical gray area to the left
    of the toolbar you want to move.
  • Drag the toolbar horizontally or vertically with your mouse
    to reposition the toolbar.

Servoy Client has two toolbars:

  • Text
  • Edit

Edit toolbar

These are the tools from the Edit toolbar:

Icon

Tool

Does this

Open Solution

Displays the Select solution dialog; the button is enabled in Data mode (edit mode).

Back

Navigates to the previous form displayed; keeps track of up to the previous nine (9) forms; the button is enabled in Data mode (edit mode) and Find mode.

Forward

Navigates to the next form after navigating to a previous form; keeps track of up to the previous nine (9) forms; the button is enabled in Data mode (edit mode) and Find mode.

New record

Creates a new record on the current form when in Data mode (edit mode); creates a new request in Find mode.

Delete record

Deletes the current record when in Data mode (edit mode); deletes the current request in Find mode.

Sort

Displays the Sort dialog when in Data mode (edit mode).
