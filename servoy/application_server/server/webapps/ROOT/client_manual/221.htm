Adding records

New records will be added to the end of the current found set.

To create a new record:

  • Choose Select > New record. OR
  • Use the keyboard shortcut CTRL-N / CMD-N. OR

  • Click on the New record tool in the Edit toolbar.

For more information about the Edit toolbar, see the section on Toolbars.
