Starting Servoy Web Client

Because Servoy Web Client is run in your web browser, there is no installation of any software.

You can start Servoy Web Client:

  • From the Servoy Server web page that shows the Select Solution list.

OR

  • By typing in the URL address for Servoy Web Client.

To launch Servoy Web Client from the Servoy Server web page:

  • Open up your web browser.
  • Enter the URL or IP address of the server.

By default, the server runs on the port 8080.

Then type "servoy-webclient".

For example:

If Servoy Server is running on a server computer called "CentralServer", then type the following text in the Address bar of your web browser:

http://centralserver:8080/servoy-webclient

You will see the following screen:

  • Click once on the solution you wish to run.
  • Servoy Web Client will open right in your web browser.

To launch Servoy Web Client by typing in a URL address:

  • Open up your web browser.
  • Enter the URL address for Servoy Web Client.

(You may need to check with your database Administrator
if you don't know the URL address for Servoy Web Client.)

  • Press the ENTER key or RETURN key on your keyboard to
    launch Servoy Web Client.
