Supported Operating Systems

The following operating systems are supported by Servoy Client:

  • Windows 98, ME, 2000, XP, Vista or higher.
  • Macintosh OS X 10.3 or higher.
  • Linux (all versions which support JDK 1.6 or higher).
  • FreeBSD (all versions which support JDK 1.6 or higher).
  • Solaris 8, Solaris 9.

You must have Java Webstart installed in order for Servoy Client to work.

NOTE: Servoy 7.0 will also work on other platforms which support JDK 1.6 or higher.
