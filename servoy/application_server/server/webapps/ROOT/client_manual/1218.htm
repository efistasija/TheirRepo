Showing omitted records

Once you have omitted one or more records, you can view all of the records you have omitted.

To show all omitted records:

  • Make sure you are in Data (ready) mode.
  • Choose Select > Show Omitted Records.

NOTE: While all the records you have omitted will be available for viewing when Show Omitted Records is chosen, you can only see more than one record displayed at a time if you are viewing the records in a list view.
