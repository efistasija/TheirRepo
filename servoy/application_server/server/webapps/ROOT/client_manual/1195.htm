Using multiple find criteria

Using multiple criteria you can perform an:

  • AND search - search on multiple fields based on multiple criteria.
  • OR search - choose to leave out records that satisfy certain criteria.

To perform an AND search:

  • Choose Select > Find. OR

Press CTRL-F / CMD-F.

  • Enter the search criteria in each field.
  • Press Enter or F3 to execute the request.

To perform an OR search:

  • Choose Select > Find. OR

Press CTRL-F / CMD-F.

  • Enter the search criteria in the field.
  • Choose Select > New record. OR

Press CTRL-N / CMD-N for each new request.

  • Enter the search criteria in the field.
  • Press Enter or F3 to execute the request.
