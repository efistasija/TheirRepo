Showing all records

You can show all records on the currently active form from:

  • Data (ready) mode.
  • Find mode.

To show all records when you are in Data (ready) mode:

  • Make sure that you are not in Find mode.
  • Choose Select > Show All.

All records on the currently active form will be included in your found set.

To show all records when you are in Find mode:

  • Make sure that you are in Find mode.
  • Choose Select > Show All.

You will be switched to Data (ready) mode on the currently active form and all records will be included in your found set.
